# Technical Documentation – Assignment 1

## 1. Technologies Used
- HTML5 for structure
- CSS3 for styling and layout
- JavaScript for interactivity

## 2. Project Structure
- index.html – Main website structure
- css/styles.css – All styling and responsive rules
- js/script.js – Theme toggle and animations
- assets/images – Project images
- docs – Documentation files

## 3. Layout & Styling
- Used Flexbox and CSS Grid for layout.
- Implemented CSS variables for theme colors.
- Organized styles into sections (Navbar, Hero, Projects, Contact, etc.).
- Used media queries to ensure responsiveness on smaller screens.

## 4. Interactivity
- Implemented a dark/light theme toggle using JavaScript and CSS variables.
- Added smooth scrolling behavior.
- Created a fade-up animation triggered on scroll using CSS transitions and JavaScript.

## 5. Responsive Design
- Tested layout using browser resizing and Developer Tools.
- Adjusted grid columns and layout spacing using media queries.
- Ensured images scale properly using responsive width settings.

## 6. Code Quality
- Removed unused CSS rules.
- Organized styles logically into sections.
- Ensured consistent indentation and formatting.
- Verified there were no console errors.
